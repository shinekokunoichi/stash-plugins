## Features
- Global search
  - Search globally on stash (title/name)
  - Supports ```@category``` search
  - Supports ```#attributes:value``` search
  - Supports multiple filters: ```@images #tags:1_female CG``` > Search for any ```images``` with ```1_female tag``` and the image title includes ```CG```
- Alphabet search
  - Available for every category
  - Note: after selecting a filter you need to double-click the next one!
