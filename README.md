# Installation
Go to `Settings > Plugins > Available Plugins > Add source`

Name: `ShinekoKunoichi`
Source URL: `https://shinekokunoichi.github.io/stash-plugins/main/index.yml`
Local Path: `ShinekoKunoichi`
